# 要望管理ツール

## アーキテクチャ

・Vue.js@2.0

・vue-router@2.0

・webpack@2.0

・Bootstrap@4.0.0

・Go@1.7

・Echo@master

・Dep@master


## Installation

```
$ go get github.com/h-tko/task-list.git

$ go get github.com/h-tko/task-list-migration.git

$ cd $GOPATH/src/github.com/h-tko/task-list-migration

$ goose up

$ cd $GOPATH/src/github.com/h-tko/task-list

$ dep ensure

$ npm install

$ webpack
```
